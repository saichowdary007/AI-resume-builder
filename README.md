# AI Resume Builder

## Description
A web application for building resumes using AI. Includes a Python backend and static frontend.

## Installation

### Local Setup
1. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the server:
   ```bash
   python main.py
   ```

### Docker Setup
1. Build and start the container:
   ```bash
   docker-compose up
   ```

## Usage
Access the application at `http://localhost:5000` after starting the server.

## Project Structure
- `main.py`: Backend server implementation
- `static/`: Frontend assets (JavaScript, CSS)
- `Dockerfile`: Docker configuration
- `docker-compose.yaml`: Docker Compose setup
- `.env`: Environment variables# AI-resume-builder
